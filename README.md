## 앱 소개

> 에디터를 이용해 쉽게 노트를 작성하고, 태그로 노트를 관리하는 애플리케이션

### 태그

  <br/>

### 노트

- 노트 생성, 수정
- 노트 내 배경 색 지정
  <br/>
  ![화면 기록 2023-12-01 오후 11 50 06](https://github.com/naringst/note-app/assets/92130993/1dc2159b-df39-4b40-a710-9271ac5c847a)

- 노트 삭제
- 노트 고정(핀 기능)
- 노트 정렬
- 노트 아카이빙
- 노트 휴지통

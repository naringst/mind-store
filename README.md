## 앱 소개

> 에디터를 이용해 쉽게 노트를 작성하고, 태그로 노트를 관리하는 애플리케이션

## 애플리케이션 기능 목록

### 태그

- 태그 생성, 삭제
- 노트 내 태그 지정

### 노트

- 노트 생성, 삭제, 수정
- 노트 내 배경 색 지정
- 노트 고정(핀 기능)
- 노트 정렬
- 노트 아카이빙
- 노트 휴지통
